How It Works: Hybrid Execution

The app uses a 3-layer validation process before taking any trade:
1. Quant Engine: Scans all MT5 pairs for technical setups (Math-only).
2. AI Brain (Llama 3.1): Validates the setup against market context and news.
3. ML Probability: Evaluates the historical success rate of similar setups.

Note: "Simulation Mode" in settings allows you to test this entire pipeline with mock data, while "Halt" or "Engage" on the dashboard controls live MT5 execution.
